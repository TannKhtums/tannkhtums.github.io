HTML/CSS Basics

HTML Basic Tags
Tag Name Description
<html> HTML This is the basic tag that will surround all of your html code.
<head> Head This element contains metadata about the document, like its title, scripts, and style sheets.
<link> Link Connects the current document and an external resource, most commonly CSS style docs
<style> Style Contains style information for the document or part of a document, contains CSS, which is applied using the <style> element.
<title> Title This shows the doc's title in the pages tab
<article> Article This element shows that it contains content that is distributed/reusable i.e. a blog post, article, product card, interactive widget, etc.

CSS Specificity

Type Selectors i.e. <p>

Class

ID

Combination of above elements

Inline styles added to an element (e.g., style="font-weight: bold;") always overwrite any styles in external stylesheets, and thus can be thought of as having the highest specificity.

!important (the Trump card)

Whichever is the most specific will override the others

Rules that appear later in the code override earlier rules if both have the same specificity.
